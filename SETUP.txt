1. Install anaconda if you do not already have it installed.
Anaconda: https://docs.anaconda.com/anaconda/install/index.html

2. Navigate to the place you downloaded this folder.You will need to use the terminal for this. 
Open Anaconda and then open the console on your computer. That will be Terminal for Mac and 
Powershell/Command Prompt for Windows. If you do not know how to do this, watch one of these
videos:
windows: https://www.youtube.com/watch?v=8-mYKkNzjU4
mac/linux: https://www.youtube.com/watch?v=KqtKD8z-NRc

3. Enter the following command into the terminal window: pip install streamlit
Let that finish before you do anything else

4. Try the following command: python3 run.py
if that doesn't work, try: streamlit run nernst_app.py
